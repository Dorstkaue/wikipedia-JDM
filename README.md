# Cultura JDM

Um projeto educativo sobre a cultura Japanese Domestic Market (JDM) com HTML semântico e CSS responsivo.

## 📋 Características

- ✅ HTML semântico e acessível
- ✅ Layout responsivo com sumário lateral (sticky)
- ✅ Navegação com ARIA labels
- ✅ Design moderno com tema dark/light
- ✅ Otimizado para mobile, tablet e desktop

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/wikipedia-JDM.git
```

2. Abra o arquivo `index.html` no seu navegador

## 📁 Estrutura do Projeto

```
wikipedia-JDM/
├── index.html      # Arquivo principal HTML
├── style.css       # Estilos CSS
└── README.md       # Este arquivo
```

## 🎨 Tecnologias

- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript (opcional para futuras melhorias)

## 📚 Conteúdo

- **História**: O movimento JDM nas décadas de 80 e 90
- **Características**: Propriedades dos carros JDM
- **Carros Icônicos**: Nissan Skyline GT-R, Toyota Supra e Mazda RX-7

## ♿ Acessibilidade

Este projeto segue boas práticas de acessibilidade web:
- Semântica HTML apropriada
- ARIA labels e roles
- Navegação por teclado completa
- Contraste de cores adequado
