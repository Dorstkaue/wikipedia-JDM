# 🏎️ Cultura JDM - Documentação Educativa

Um projeto educativo e interativo sobre a cultura **Japanese Domestic Market (JDM)**, desenvolvido com foco em **semântica HTML** e **design responsivo acessível**. Este projeto demonstra boas práticas modernas de desenvolvimento web frontend.

> **Nota**: Este é um projeto de fins educacionais criado para aprender padrões web semânticos e acessibilidade.

---

## 📋 O Que é JDM?

A cultura **JDM (Japanese Domestic Market)** refere-se aos veículos e modificações desenvolvidos para o mercado japonês. É um movimento apaixonante entre entusiastas automotivos que celebra a engenharia, criatividade e paixão por carros de alto desempenho.

---

## ✨ Características Principais

### 🏗️ Estrutura Semântica
- ✅ Uso correto de tags HTML5 semânticas (`<header>`, `<main>`, `<section>`, `<aside>`, `<footer>`)
- ✅ Hierarquia de títulos bem estruturada
- ✅ Elementos semânticos para navegação e conteúdo

### 🎨 Design Responsivo
- ✅ Layout fluido com **CSS Grid** (2 colunas em desktop, 1 em mobile)
- ✅ Sumário lateral **sticky** que acompanha a leitura
- ✅ Totalmente otimizado para **mobile, tablet e desktop**
- ✅ Design moderno com paleta de cores harmoniosa

### ♿ Acessibilidade (WCAG)
- ✅ **ARIA labels** e **roles** adequados para leitores de tela
- ✅ Navegação completa por **teclado** (Tab, Enter)
- ✅ Contraste de cores em conformidade com padrões
- ✅ Elementos focáveis com indicadores visuais claros
- ✅ Estrutura lógica e semântica

### 🚀 Performance
- ✅ Sem dependências externas (apenas HTML + CSS puro)
- ✅ Carregamento rápido
- ✅ Scroll behavior suave

## ♿ Padrões de Acessibilidade (WCAG 2.1)

Este projeto implementa as seguintes práticas recomendadas:

✅ **Semântica HTML apropriada**
- Uso de headers, mains, sections, articles e footers
- Títulos hierárquicos corretos (h1 > h2 > h3)

✅ **ARIA (Accessible Rich Internet Applications)**
- `role="navigation"` para areas de navegação
- `role="main"` para conteúdo principal
- `role="complementary"` para sidebars
- `aria-label` e `aria-labelledby` para contexto

✅ **Navegação por Teclado**
- Todos os links e botões são acessíveis via Tab
- Foco visual claro em elementos interativos

✅ **Contraste de Cores**
- Razão de contraste ≥ 4.5:1 para texto e fundo

✅ **Responsividade**
- Funciona em qualquer tamanho de tela
- Touch-friendly em dispositivos móveis
